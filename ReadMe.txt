Title: Place a Hat, Mush and a Shade Artificially on Your Face

Dependencies:

OpenCV

How to run:

1. From 'images' folder select the image and put it's path in the .cpp

2. Open a terminal and compile and execute with the following:

   g++ Mush.cpp -o Mush `pkg-config --cflags --libs opencv`
   ./Mush
